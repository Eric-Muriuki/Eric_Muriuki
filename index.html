Lab Challenges

HTB Academy: Introduction to Network Traffic Analysis

Completed: November 2023 Traffic Analysis Duration: 6 hours

Problem Statement

Analyze network traffic captures to identify communication patterns, extract files from network streams, and detect malicious activities.

Approach

Used Wireshark to examine packet captures, applied filters to isolate specific protocols and conversations, followed TCP streams to reconstruct data exchanges, and used built-in tools to export objects from network traffic.

Tools Used

Wireshark, NetworkMiner, tcpdump

Key Lessons

Learned to effectively use display and capture filters in Wireshark, understand network protocol behavior, identify anomalies in traffic patterns, and extract files from network streams for further analysis.

TryHackMe: DNS in Detail

Completed: October 2023 DNS Security Duration: 3 hours

Problem Statement

Understand DNS protocol workings, record types, and common vulnerabilities while performing practical DNS reconnaissance and analysis.

Approach

Used command-line tools like nslookup and dig to query DNS records, examined different record types (A, AAAA, CNAME, MX, TXT), understood DNS hierarchy and resolution process, and explored DNS security considerations.

Tools Used

nslookup, dig, Wireshark

Key Lessons

Gained deep understanding of DNS protocol, record types, and their purposes; learned to perform DNS reconnaissance; understood DNS security issues like cache poisoning and how DNSSEC helps mitigate them.

HTB Academy: Web Requests

Completed: December 2023 Web Security Duration: 5 hours

Problem Statement

Understand HTTP protocol, request methods, headers, and status codes while analyzing web traffic and identifying security-relevant information.

Approach

Used browser developer tools to inspect web requests, crafted custom HTTP requests with curl, analyzed request and response headers for security information, and understood different HTTP methods and their security implications.

Tools Used

Browser Developer Tools, curl, Burp Suite

Key Lessons

Mastered HTTP protocol details, understood security headers and their importance, learned to manipulate web requests for testing purposes, and gained insight into web application communication patterns that are essential for security testing.
