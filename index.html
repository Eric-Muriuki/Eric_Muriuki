Lab 2 — Binary Exploit (Buffer Overflow) (Intermediate)
Problem statement

Exploit a vulnerable C binary to gain a shell on the lab VM.

Approach

Static analysis with Ghidra, identify vulnerable function, craft ROP chain to bypass NX/ASLR where required, and test on local VM.

Tools used
Ghidra, pwntools, gdb, QEMU
Screenshots
Key lessons learned
  • Careful memory layout analysis is required; debug symbols accelerate the process.
  • Modern mitigations require creative chaining and fallback strategies.
Lab 3 — Network Forensics (Pcap Analysis) (Intermediate)
Problem statement

Analyze a provided pcap to identify exfiltrated files and the C2 infrastructure.

Approach

Use Wireshark to filter suspicious flows, extract files from HTTP/SMTP, and map IPs to domains using passive DNS and whois.

Tools used
Wireshark, tcpdump, networkx, python (scapy)
Screenshots
Key lessons learned
  • Effective filtering reduces noise; scripting helps reproducibility.
  • Documenting the analysis steps is essential for later handoff.
